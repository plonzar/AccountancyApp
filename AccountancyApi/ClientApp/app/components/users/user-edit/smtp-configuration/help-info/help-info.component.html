Konfiguracja SMTP

Pozwala na wysyłanie wiadomości email do Twoich klientów.

Nie musisz konfigurować tych ustawień, oprogramwanie posiada już skonfigurowane ustawienia, tak aby klieci mogli odbierać wiadomości.

Każda wiadomość wysłana z tego serwisu jest podpisywana Twoją nazwą firmy. Jeśli klient będzie chciał odpowiedzieć na Twojego maila, zostanie on przekierownany na email, który był podczas rejestracji.

Jeżeli chcesz skonfigurować te ustawienia, dowiedz się kto jest Twoim dostawcą usług pocztowych i jakiej konfiguracji należy użyć, aby zintegrować się z pocztą. W przypadku gdy podany przez Ciebie port nie będzie działał zostanie ustawiony domyślny port 587, jeśli na nim pojawi się błąd program wyświetli komunikat o nie udanej próbie konfiguracji. Niektóre serwisy taki jak Gmail, Yahoo wymagają aby włączyć w ustawieniach dostęp do poczty dla mniej bezpiecznych aplikacji. Poniżej podano adresy do konfiguracji do kilku popularnych serwisów pocztowych:

  • Gmail
  • Yahoo
  • Microsoft Outlook
  • Home
  • Wirtualna Polska / O2
  • Onet

Jeżeli chcesz aby system spróbował automatycznie skonfigurować ustawienia SMTP przejdź do Autokonfiguracji.
